INDUCTOR(T3) — “전류 변화/자기장 = 축적 → 방출”로 딜을 만든다
1) 플럭스 차지(Flux Charge) → 과부하 방출(Overdrive Burst)
평타/이동/피격으로 “플럭스(자기장 에너지)” 스택이 차오름.
스킬 사용 시 스택을 소비해 **짧은 시간 ‘Hit CD 감소 + Orb 속도/관통 + 추가타’**로 폭발.
포인트: 평소엔 무난, **각 잡으면 순간 딜이 터져서 ‘근접인데도 위협’**이 됨.

추천 효과 예시
플럭스 10스택 소비: 2.5초 동안 Hit CD 750ms → 450ms, Orb가 짧게 관통(1회) + 적중 시 미니 넉백

2) 와전류 트랩(Eddy Current Trap)
바닥에 자기장 링(원형 장판) 설치.
링 안의 적은 이동속도 감소 + “방어막/쉴드가 있을수록” 추가 피해(현실에서 도체에 유도전류 생기는 이미지).
포인트: 인덕터의 “자기인력 약함”을 지역 통제+근접전 유리로 바꿔줌.

3) 코일 아크(Inductive Arc) — 오브 간 ‘전기 아크’로 준-원거리 딜
오브가 여러 개(4개)인 장점을 살려서,
오브가 적 근처에 모이면 오브↔오브 / 오브↔적 사이에 짧은 아크가 연결되어 추가 틱딜.
포인트: 인덕터가 “근접만”이 아니라, 붙기 직전/붙은 후 둘 다 위협적인 딜 형태가 됨.

4) 렌츠 카운터(Lenz Counter) — 맞을수록 반격이 강해지는 반격기
일정 피해를 짧은 시간에 받으면 역자기 펄스가 터져 주변에 피해 + 밀쳐냄.
포인트: 브루저 느낌 확 살고, 근접전에서 “들어가면 손해” 압박이 생김.


TRANSFORMER(T3) — “승압/강압, 결합, 절연”을 공격으로 바꾼다
1) 승압 펀치(Step-Up Strike) — 저장한 에너지를 ‘한 방’으로
주변(아군 포함)에서 발생한 **타격/전류(=전투 이벤트)**를 조금씩 저장했다가,
스킬로 짧은 사거리의 부채꼴/직선 빔으로 고폭발 피해.
포인트: DPS는 낮아도 “궁처럼 한 번은 무섭게” 만들 수 있음.
(트랜스는 원래 “연속딜”보다 “변환/증폭” 이미지가 강함)

2) 탭 체인(Coupling Chain) — ‘결합’해서 추가타를 만든다
적 하나를 “1차 코일 마킹”하면, 이후 일정 시간 동안
오브가 맞출 때마다 근처 다른 적에게 2차 타격(감쇠된 번개/링크) 발생.
포인트: 1:1 약한 대신 다수전에서 존재감이 생김.

3) 모드 스왑: 강압(탱) ↔ 승압(딜)
버튼 한 번으로 두 모드를 전환(쿨 짧게).
강압 모드: 받는 피해↓ / 쉴드 회복↑ / 딜↓
승압 모드: 쉴드 일부를 소모하면서 Orb DMG↑ 또는 Hit CD↓
포인트: 트랜스포머는 “유틸 탱” 정체성 유지하면서도, 필요할 때 공격적으로 변신 가능.

4) 절연 필드(Isolation Field) — 방어기가 곧 공격이 되는 형태
짧게 절연 돔 생성: 바깥에서 안으로 투사체 일부 무효(또는 피해 감소).
돔이 끝날 때 내부 적에게 누적된 에너지로 방출 피해.
포인트: “팀 유틸”이면서도 전투 기여가 수치로 체감됨.
지금 스탯 기준으로 “가성비” 좋은 추천 조합 (바로 강해짐 체감)
INDUCTOR: 플럭스 과부하(버스트 창) + 와전류 트랩(장판)
→ 근접 약점을 “각 잡으면 터지는 캐리”로 해결
TRANSFORMER: 승압 펀치(한 방) + 모드 스왑(탱↔딜)
→ 평소엔 탱/유틸, 한타/추격에서 “킬각”을 직접 만들 수 있게 됨
